# 🛠️ Manual Override

Welcome to my daily code journal. This repository exists because I am tired of letting AI drive the car while I just give it directions. I am officially stepping away from the "prompt shield" and forcing myself to build raw syntax muscle memory—even if that means writing incredibly boring stuff every single day. 

Expect unoptimized logic, syntax errors, and a lot of mundane scripts as I learn to talk to the computer by myself.

## 📅 Daily Progress Log

### Day 1: The Ugly Pixel Cat
* **What I did:** Wrote a multi-dimensional matrix-based grid in pure Python to render a pixel cat in the terminal. No prompt shortcuts allowed.
* **What I learned:** How 2D arrays/lists map directly to pixel coordinates (which is actually a foundational concept for Computer Vision!). Also brushed up on using `end=""` in Python print functions to keep characters on the same line.
* **The Reality:** It's supposed to be a majestic kitten, but honestly, it turned out pretty ugly. It looks back at me from the terminal with a mix of judgment and pity, but hey—**I wrote every single line myself.**
